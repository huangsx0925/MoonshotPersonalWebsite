减法法则

设计方法使用流程


1.列出产品的组件 List product components


2.去除一个重要的部分 Subtract one essential component


3.想象出“虚拟产品” Visualize the resulting form - virtual product


4.找出在“这种情况下”的好处、市场、潜在顾客 Identify benefits, markets and potential clients “in this situation”


5.找出可能的挑战，及如何克服挑战的方法 Identify possible challenges and ways to overcome them


6.做出必要的调整 Make necessary adaptations
除法法则

设计方法使用流程


1. 对产品或服务列出组成部分的列表 Make a list of all product/service components


2. 尝试在时间或空间上重新给这些组成部分排序。 Attempt to rearrange the components in time or in space


3. 想象这个虚拟的产品会是怎样 Visualize the Virtual Product


4. 想它的好处，进行市场筛选 Understand the benefits apply market filter


5. 想想挑战，进行可行性筛选 Challenges ，apply implementation filter


6. 做出必要的调整 Make necessary adaptations
加法法则

设计方法使用流程


1. 列出封闭世界的组件 List the Closed World components


2. 定义一个清晰明确的任务 Define a clear mission


3. 从封闭世界中选择一个组件 Select a component from the Closed World


4. 试想这个组件可以如何帮我们完成这个任务 Imagine how this component can help us accomplish this task


5. 想象得到的“虚拟情境” Imagine the "virtual situation"


6. 发现潜在的价值 Find potential benefits


7. 识别可行性挑战，对想法进行改进 Identify feasibility challenges and improve ideas
